idea使用git托管代码

发表于 2021-01-16 分类于 环境搭建

idea中使用git来管理我们的代码

在gitee中拥有一个仓库

在gitee上建立一个自己的仓库
如下图

仓库建好之后如下图

现在的仓库中没有任何代码，下面去idea中编写代码并托管到git上吧

在idea中配置git

新建一个Java项目，项目名最好和仓库名一致

在idea中配置git

依次点击File –> Settings在打开的窗口中依次找到Version Control –> git 或者直接搜索git

上图为在idea中配置git，这一步很重要，如果没有配置成功，无法使用git功能

将代码托管到git仓库中

按照上图依次点击，选择刚才创建的Java项目作为git仓库，到这一步我们的本地仓库就已经建好了，现在只要把代码上传就完成了代码托管。

代码上传

第一步、按照上图选中项目右键点击选择git选择add

第二步、按照上图选中项目右键点击选择git选择Commit Directory进入如下界面

点击commit后可能会出现如下图所示窗口，不用理会

到现在为止，我们的代码已经提交到了本地的仓库，文件的颜色已经全部正常，现在我们只要把文件提交到git仓库就可以了。

右键点击项目 –> Git –> Repository –> Push

进入你刚才创建的仓库，会看见下图，就是你的仓库地址

填写完仓库地址，点击ok，如果是第一次使用会弹出账号密码窗口，填写自己的gitee的账号和密码，确认就可以了，最好不要填错，填错账号密码之后怎么解决，本文不会写解决方法。如果以前填写过账号和密码就不会弹出填写账号和密码界面。填写完成之后会出现如下界面。

点击之后就会提示，代码已经提交到了远程仓库。

拉取代码

右键点击项目 –> Git –> Repository –> Pull

出现拉取界面，直接点击pull拉取代码就可以了，如果有分支，注意分支问题

总结

流程：
1、在gitee中创建一个仓库
2、在idea中新建一个Java项目，项目名最好和仓库名一致
3、在idea中配置git（如果已经配置过则不需要）
4、点击VCS –> import into Version Control –> Create Git Repository 选择刚才创建的项目，作为本地仓库
5、右键点击项目 –> Git –> add
6、右键点击项目 –> Git –> Commit Directory 填写好提交信息，进行提交，将代码提交到本地仓库
7、右键点击项目 –> Git –> Repository –> Push
8、在弹出的界面中点击左上角的 Define remote 定义远程
9、填写gitee仓库地址
10、填写gitee账号和密码（如果以前填写过则没有这一步）
11、选择自己刚才提交的提交信息（代表了这一次提交），进行提交，提交成功之后，代码就在远程仓库了

注意：
1、在上传代码的过程中，如果你只想上传一个文件夹（包）你需要在包里面新建一个文件，因为不允许空文件夹上传。
2、账号密码一定要填写正确，一旦填写错误自行寻找解决办法。
